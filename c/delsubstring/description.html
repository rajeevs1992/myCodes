Substring Killer

You will be given 2 strings and your task is to delete the second string from the first and give the modified string.

Input


The first line of the input will contain the number of testcases.
The next line of the input will contain the main string.
The next line will contain the string to be deleted.

Length of string,l, will be less than 1000.
Length of the string to be deleted will be <=l


Same repeats for n testcases,n<10

Output

For each test case, output the string after deletetion

Example

Input
2
hello
ll
welcome
el
Output
heo
wcome
